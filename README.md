# Ryubing Issue Tracker
This repository is the dedicated issue tracker for the [Ryubing](https://ryujinx.app) project.

[Click here](https://github.com/Ryubing/Issues/issues/new/choose) to open an issue.


# Rules
- Issues that contain or link to piracy in any way will be deleted and the poster will be blocked *from the entire [Ryubing](https://github.com/Ryubing) organization*.
- Issues that are improperly filed or missing required information will be closed. Please fill out all of the fields.
- Issues created/written with the help of LLMs (AI) will be closed, and the poster will be blocked *from the entire [Ryubing](https://github.com/Ryubing) organization*.
